AI Decision-Making Coach

Project Overview

AI Decision-Making Coach is a simple web application that helps users compare two options and make better decisions using a scoring-based evaluation method. Users enter two choices and assign scores to each option. The system analyzes the scores and recommends the better choice.

Objectives

- Assist users in making informed decisions.
- Compare multiple options using a simple scoring technique.
- Demonstrate the use of AI-inspired decision support systems.
- Provide an easy-to-use web interface.

Features

- Compare two decision options.
- Score-based analysis.
- Instant recommendation.
- Simple and user-friendly interface.
- Lightweight Flask backend.

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

Installation

1. Clone the repository:

git clone https://github.com/your-username/AI-Decision-Coach.git

2. Navigate to the project folder:

cd AI-Decision-Coach

3. Install dependencies:

pip install -r requirements.txt

4. Run the application:

python app.py

5. Open your browser and visit:

http://127.0.0.1:5000

Working Principle

The application accepts two decision options and their corresponding scores. The backend compares the scores and determines which option is more suitable. The result is displayed instantly to the user.
